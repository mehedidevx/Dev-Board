Question-1: What are the different ways to select an element in the DOM?

1. `document.getElementById("id")` – Selects an element by its ID.
2. `document.getElementsByClassName("class")` – Selects elements by class name.
3. `document.getElementsByTagName("tag")` – Selects elements by tag name.
4. `document.querySelector("selector")` – Selects the first matching element using a CSS selector.
5. `document.querySelectorAll("selector")` – Selects all matching elements using a CSS selector.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML: Gets or sets the HTML content inside an element.
innerText: Gets or sets only the visible text inside an element (ignores hidden elements).
textContent: Gets or sets all text content, including hidden elements (ignores HTML).

Question-3: What is event delegation in the DOM?

A technique where a parent element listens for events on its child elements using event bubbling, reducing the need for multiple event listeners.

Question-4: What is event bubbling in the DOM?

When an event on a child element bubbles up to its parent elements in the DOM hierarchy, allowing them to respond to the same event.

Question-5: Create, Add, and Remove Elements using JavaScript:

Create an element: `const el = document.createElement("div");`
Add an element: `parent.appendChild(el);` or `parent.append(el);`
Remove an element: `parent.removeChild(el);` or `el.remove();`
